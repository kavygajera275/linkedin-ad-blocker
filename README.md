# LinkedIn Ad Blocker 🚀

A professional Chrome extension that blocks:

- Sponsored LinkedIn posts
- Promoted jobs
- Sidebar ads
- Tracking ads

## Features

✅ ON/OFF Toggle  
✅ Live Ad Counter  
✅ Professional UI  
✅ Fast Performance  
✅ Chrome Extension Manifest V3

## Installation

1. Download project
2. Open Chrome
3. Go to chrome://extensions/
4. Enable Developer Mode
5. Click Load Unpacked
6. Select project folder

## Technologies Used

- JavaScript
- HTML
- CSS
- Chrome Extensions API
